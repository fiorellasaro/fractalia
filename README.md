# Fractalia üåÄ

**Fractalia** is an interactive web application that allows users to explore, generate, and export mesmerizing 3D geometric fractals. Built with modern web technologies, it brings the mathematical beauty of self-similarity to your screen with real-time performance.

## üåü Features

- **Interactive 3D Rendering**: Explore fractals in a fully navigable 3D space using React Three Fiber.
- **Multiple Geometry Types**: Generate classic fractals like the Menger Sponge, Sierpinski Tetrahedron, and custom Cubic/Sphere variations.
- **Radial & Spiro Mode**: Create complex radial patterns with the Spiro Fractal generator.
- **Real-time Customization**:
  - Adjust recursion depth, scale, and spacing.
  - Modify materials (Metalness, Roughness, Color).
  - Control lighting and environment settings.
- **Export & Share**:
  - **Story Mode**: Export your creations as 9:16 vertical images or videos, perfect for social media stories (Instagram, TikTok).
  - **High Quality**: Downloads high-resolution PNGs and WebM videos.
- **Responsive Design**: Fully optimized for both desktop and mobile devices.

## üåø Fractals in Nature

Fractals are not just abstract mathematical concepts; they are the geometry of nature. Many natural objects exhibit **self-similarity**, where smaller parts resemble the whole.

### 1. Ferns

Ferns are a classic example of self-similarity in nature. Each frond is composed of smaller sub-fronds (pinnae), which are in turn composed of even smaller leaflets (pinnules), all mimicking the shape of the entire frond.

![Barnsley Fern](https://upload.wikimedia.org/wikipedia/commons/thumb/a/a5/Barnsley_Fern_fractals_-_4_states.PNG/400px-Barnsley_Fern_fractals_-_4_states.PNG)
_The Barnsley fern, a mathematical fractal that mimics natural ferns._

### 2. Romanesco Broccoli

Perhaps the most visually striking edible fractal, the Romanesco broccoli displays a logarithmic spiral structure. Each bud is composed of a series of smaller buds arranged in a spiral, continuing down to microscopic levels.

![Romanesco Broccoli](https://upload.wikimedia.org/wikipedia/commons/thumb/4/4f/Romanesco_broccoli_%282%29.jpg/400px-Romanesco_broccoli_%282%29.jpg)
_The self-similar spiral structure of Romanesco broccoli._

### 3. Snowflakes

Snowflakes form via crystallization patterns that branch out symmetrically. While not infinite like mathematical fractals, their branching arms (dendrites) show recursive growth patterns similar to the Koch Snowflake.

![Snowflake](https://upload.wikimedia.org/wikipedia/commons/thumb/f/f5/Snowflakes_Wilson_Bentley.jpg/400px-Snowflakes_Wilson_Bentley.jpg)
_Complex branching patterns in real snowflakes._

## üöÄ Getting Started

To run this project locally:

1.  **Clone the repository**:

    ```bash
    git clone https://github.com/yourusername/fractalia.git
    cd fractalia
    ```

2.  **Install dependencies**:

    ```bash
    npm install
    ```

3.  **Run the development server**:

    ```bash
    npm run dev
    ```

4.  **Build for production**:
    ```bash
    npm run build
    ```
